# AI-Powered Task Management System

## 📌 Project Overview

The AI-Powered Task Management System is a machine learning and NLP-based application that automatically classifies tasks, predicts their priority, and helps distribute workloads efficiently.

---

## 🎯 Objectives

- Automatically classify tasks
- Predict task priority
- Balance workloads
- Improve productivity
- Provide an interactive dashboard

---

## 🚀 Features

- NLP-based Task Classification
- Task Priority Prediction
- Workload Balancing
- Interactive Streamlit Dashboard
- Machine Learning Model Comparison
- Performance Evaluation

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- Streamlit
- XGBoost

---

## 🤖 Machine Learning Models

- Naive Bayes
- Support Vector Machine (SVM)
- Random Forest
- XGBoost
